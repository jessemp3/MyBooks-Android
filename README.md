# MyBooks-Android

Projeto Android desenvolvido em Java para gerenciar e visualizar livros.

## Funcionalidades

- Cadastro de livros(dentro do código, por enquanto)
- Visualização de detalhes dos livros
- tela de favoritos
- Organização da biblioteca pessoal

## Tecnologias Utilizadas

- Android SDK
- Java
- retrofit
- room

## Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/jessemp3/MyBooks-Android.git
   ```
2. Abra o projeto no Android Studio.
3. Compile e execute em um dispositivo ou emulador Android.
4. ou baixe a versão APK
